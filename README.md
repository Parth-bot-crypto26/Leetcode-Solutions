# LeetCode Daily Solutions & Explanations

This repository is a public archive dedicated to tracking and documenting my daily progress on LeetCode. Its main purpose is to host clean, working code implementations alongside detailed breakdowns of the thought processes, patterns, and complexities behind each solution.

---

## 📂 What This Repository Contains

Every problem I solve has its own dedicated folder containing two specific files:
1. **`solution.cpp`**: The complete, clean C++ implementation of the solution.
2. **`explanation.txt`**: A text file written in my own words explaining the exact approach used, edge cases handled, a manual step-by-step dry run of the logic, and the formal Time and Space complexity analysis (Big-O).

---

## 🛠️ Compilation & Local Execution

The solutions are written in standard modern C++. You can compile and run any of the `.cpp` files locally via your terminal using `g++`:

```bash
g++ -std=c++11 solution.cpp && ./a.out
